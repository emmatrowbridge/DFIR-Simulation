# Investigation Questions
---
Answer the following questions using only the provided evidence.

All answers must be directly supported by artifacts.

---
## Initial Access and File Findings
1. What is the exact file name of the downloaded document relevant to the investigation?
2. Where on the system was the file located?
3. Which artifact confirms that the file was downloaded from the web?
4. What type of file was executed?

## Process Execution Findings
5. What application was used to open the file?
6. What specific user action enabled execution of the document?
7. What process was spawned as a result of opening the file?
8. Which artifact best supports the parent-child relationship between the document and the spawned process?

## Post-Execution Activity
9. What observable activity occurred after the document was executed?
10. Which temporary or log files were created as a result of the execution?
11. Why did the activity not appear in the Windows Security logs?

## Correlation and Incident Analysis
12. Using multiple artifacts, reconstruct the most likely sequence of user and system actions that occurred from the moment the attachment was accessed to the final observable system activity.
13. Which combination of artifacts most strongly supports that this activity represents a phishing-based compromise rather than normal onboarding behavior, and why?
14. Which artifacts distinguish legitimate onboarding activity from malicious activity?

---
## Instructions
* Use exact file names and paths where applicable
* Support all answers with evidence from the artifacts
* Some questions require combining multiple sources of data
* Do not include assumptions not supported by evidence
