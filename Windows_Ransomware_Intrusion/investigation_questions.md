# Investigation Questions
---
Answer the following questions using only the provided evidence.

All answers must be directly supported by artifacts.

---
## Account and System Access
1. Which account shows evidence of successful logon activity on the system?
2. What logon type is associated with this activity?
3. What is the timestamp of the earliest relevant logon event for this account?
4. What is the Logon ID associated with this session?
5. Which artifact provides evidence of session creation beyond standard logon events?
6. Based on the evidence, does this activity most likely originate from a local or remote session? Justify your answer using artifact data.

## Program Execution
7. Which executable is associated with suspicious activity on the system?
8. Which artifact provides evidence of this execution?
9. What is the full path of this executable?
10. Approximately how many times was this executable run?
11. What is the timestamp of the last recorded execution of this program?
12. Identify another executable that was run in close temporal proximity. Provide supporting evidence.

## Filesystem Impact
13. What file extension is associated with modified files on the system?
14. Identify four files that exhibit this extension.
15. Identify at least three different directories containing affected files.
16. Based on the distribution of affected files, does the modification appear targeted or widespread? Support your answer with evidence.
17. Are there any files that do NOT appear to be affected? Provide one example and explain why it may have been excluded.

## Ransom Note
18. What is the name of the file containing the message left on the system?
19. Provide one exact sentence from this message.
20. What does the message indicate about the user's ability to recover files?
21. Does the message include any instructions or warnings? If so, describe them.

## Correlation and Root Cause Analysis
22. Based on timestamps and artifacts, what is the likely sequence of events from initial access to file modification?
23. What evidence suggests how the system was initially accessed?
24. What evidence suggests how the malicious activity was executed?
25. What evidence indicates that the activity affected multiple parts of the system?

## Impact Assessment
26. Based on the artifacts, how widespread is the impact on the system?
27. What types of files were affected? Provide at least three distinct file types.
28. What evidence suggests that this activity may have impacted other users or systems?

---
## Instructions
* Use exact file names, paths, timestamps, and values where applicable
* Support all answers with evidence from the artifacts
* Some questions require combining multiple sources of data
* Do not include assumptions not supported by evidence
